생성기로 돌아가기

GDS 엔트리 생성기 사용 메뉴얼

1. 소개

GDS 엔트리 생성기는 여행사 ERP 시스템에서 복사한 탑승객 정보를 GDS(Global Distribution System)에서 사용하는 표준 명령어 형식으로 자동 변환해주는 도구입니다.

수작업으로 발생할 수 있는 오류를 줄이고, 항공권 예약 및 발권(PNR 생성) 업무의 효율성을 높이기 위해 제작되었습니다. 특히 유아(INF)의 개월 수 자동 계산 등 편의 기능을 제공합니다.

2. 사용 방법

Step 1: 탑승객 정보 입력

ERP 시스템의 예약일행 화면에서 CTRL+Q를 눌러 복사 모드를 활성화한 후, 탑승객 정보를 복사하여 생성기 화면의 표에 그대로 붙여넣습니다. (Ctrl+V)

또는, 아래 '입력 데이터 상세 가이드'를 참고하여 표에 직접 정보를 입력할 수도 있습니다. 필수 항목(*)은 반드시 입력해야 합니다.

Step 2: 유아(INF) 동반 시 출발일 입력

입력된 정보에 '탑승객 유형'이 INF인 유아가 포함된 경우, 정확한 개월 수 계산을 위해 여행 출발일 입력 섹션이 나타납니다.

YYYY-MM-DD 또는 YYYYMMDD 형식으로 출발일을 입력하면, 생성기가 자동으로 유아의 생년월일을 기준으로 만 개월 수를 계산하여 세이버(Sabre) 엔트리에 반영합니다.

Step 3: 엔트리 생성 및 결과 확인

정보 입력이 완료되면 엔트리 생성 버튼을 클릭합니다. 시스템은 입력된 값의 유효성을 검사하며, 오류가 있을 경우 해당 행과 필요한 정보를 알려줍니다.

성공적으로 처리되면, 하단에 '항공권 NM 형식'과 '세이버(Sabre) 엔트리'가 생성된 것을 확인할 수 있습니다.

Step 4: 결과 복사

각 결과 창의 우측 상단에 있는 복사 버튼을 클릭하여 생성된 엔트리를 클립보드에 복사할 수 있습니다. 복사된 엔트리는 GDS 시스템에 바로 붙여넣어 사용할 수 있습니다.

3. 입력 데이터 상세 가이드

표에 입력하는 데이터는 아래의 형식과 규칙을 따라야 합니다. *표시는 필수 입력 항목입니다.

항목 설명 입력 형식 / 자동 변환 예시
한글명 참고용 정보이며, 실제 엔트리 생성에는 사용되지 않습니다. - 홍길동
영문 성* 여권에 기재된 영문 성(Last Name)입니다. KIM HONG
영문 이름* 여권에 기재된 영문 이름(First Name)입니다. GILDONG GILDONG
나이* 만 나이입니다. 숫자만 입력 30
생년월일* 8자리 생년월일입니다. YYYYMMDD 형식
(예: 1990-12-25 → 19901225로 자동 변환)		 19901225
성별* 남/여 또는 M/F로 입력합니다. M 또는 F
(예: 남 → M, 여 → F로 자동 변환)		 M
탑승객 유형* 성인, 소아, 유아를 구분합니다. ADT, CHD, INF 중 하나
(예: 성인→ADT, 소아→CHD 자동 변환)		 ADT

4. 자주 묻는 질문 (FAQ)

Q. '엔트리 생성' 버튼을 눌렀는데 "입력 오류" 알림이 뜹니다.
A. 알림 메시지에 표시된 행의 필수 항목들이 올바른 형식으로 모두 입력되었는지 확인해주세요. 예를 들어 생년월일이 8자리가 아니거나, 나이가 숫자가 아닌 경우 오류가 발생합니다.

Q. 유아(INF)가 없는데도 출발일을 입력해야 하나요?
A. 아닙니다. 출발일 입력 섹션은 탑승객 중에 '유형'이 INF인 승객이 있을 경우에만 나타납니다.

Q. 표의 행이 부족하면 어떻게 하나요?
A. 표의 마지막 행에 내용을 입력하면 자동으로 새로운 행이 추가됩니다. 필요한 만큼 행을 추가하여 사용할 수 있습니다.
